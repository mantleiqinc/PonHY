About the Thermodynamic Database

The PoNHy thermodynamic database is stored as Zarr arrays in DB0/{rock_code}_all/. Each array has dimensions pressure (10–40,010 bar), temperature (0–1450 °C), and W/R ratio (0–0.2), and stores 19 thermodynamic fields including mol_kg_H2, rho_H2, V_H2, and solubility metrics. Two rock lithologies are included: HZ1 (Harzburgite) and LH1 (Lherzolite). The chart above uses the mol_kg_H2 field at a fixed pressure of ~51 MPa. 

import xarray as xr
ds  = xr.open_zarr("Data_Pyrenees/DB0/LH1_all/LH1_xarray_glpro_all.zarr")
da  = ds["__xarray_dataarray_variable__"]   # dims: (pressure, temperature, w2r, fields)
h2  = da.sel(pressure=5000, method="nearest",   # ~500 MPa
             temperature=300, method="nearest",
             w2r=0.16, fields="mol_kg_H2")
print(f"H₂ at 300°C / 500 MPa / W:R=0.16 → {float(h2.values):.4f} mol/kg")

How PoNHy computes serpentinization degree

PoNHy uses a pre-computed lookup table that maps joint density–susceptibility pairs to serpentinization degree. The table spans from 100% serpentinized (ρ ≈ 2.62 g/cm³, χ ≈ 0.07 SI) to 0% (ρ ≈ 3.24 g/cm³, χ ≈ 0.001 SI), representing the progressive alteration of olivine-bearing mantle rocks (Harzburgite/Lherzolite) to serpentine minerals. This degree is then used to apply a correction factor to the H₂ production rates from the thermodynamic database. 

# From ponhy.py — serpentinization data table
serpentinization_data = {

    'y_points': np.array([0.07, 0.0682775, ...]), # susceptibility (SI)

The KK-PR Equation of State (from PoNHy utils/general.py)
def compute_h2_solubility_kk_pr(p_mpa, t_c):
    R    = 83.14472          # cm³·bar/(mol·K)
    T_K  = t_c + 273.15
    P_bar = p_mpa * 10.0
    # H₂ critical properties
    Tc, Pc, omega = 33.19, 13.13, -0.216
    Tr    = T_K / Tc
    alpha = (1 + (0.37464 + 1.54226*omega - 0.26992*omega**2) * (1 - np.sqrt(Tr)))**2
    a     = 0.45724 * (R*Tc)**2 / Pc * alpha
    b     = 0.07780 * R * Tc / Pc
    A, B  = a*P_bar/(R*T_K)**2, b*P_bar/(R*T_K)
    # Cubic: Z³ - (1-B)Z² + (A-3B²-2B)Z - (AB-B²-B³) = 0
    roots = np.roots([1, -(1-B), (A-3*B**2-2*B), -(A*B-B**2-B**3)])
    Z     = max(r.real for r in roots if r.imag == 0 and r.real > 0)
    ln_phi = Z - 1 - np.log(Z-B) - A/(2.828*B) * np.log((Z+2.414*B)/(Z-0.414*B))
    fugacity = P_bar * np.exp(ln_phi)
    kH    = np.exp(-4.7982 + 6.3536/Tr + 9.0*np.log(Tr))   # bar
    return max(fugacity / kH, 0.0)   # mol/kg H₂O
